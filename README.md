
Aplicativo mobile para gerenciamento de tarefas, desenvolvido com React Native e Expo.



- React Native
- Expo (SDK 54)
- React Navigation
- Axios



- ✅ Listar tarefas
- ✅ Criar nova tarefa
- ✅ Editar tarefa
- ✅ Marcar como concluída
- ✅ Excluir tarefa





```bash
git clone https://github.com/rafasilvc/tasks-app.git
cd tasks-app
```



```bash
npm install
```



```bash
npx expo start --tunnel
```

Escaneie o QR Code com o app **Expo Go** no celular.
